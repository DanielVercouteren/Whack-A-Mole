# Whack-A-Mole
Whack a Mole - IMTHMI

## Doel
>De eindopdracht bestaat uit het bouwen van een prototype van een HMI. Als input voor de eindopdracht kiest de
>groep een ontwerp van een individueel lid. De beoordelaar van het vak stuurt hierin bij zodat de benodigde
>(hardware) componenten ook daadwerkelijk beschikbaar zijn.
>
>Het prototype moet een werkende versie van het ontwerp zijn, waarop testen uitgevoerd kunnen worden. Deze
>testen hebben tot doel om te kunnen concluderen of het HMI-concept wel, of niet, levensvatbaar is. Het
>uitgangspunt voor het prototype is de verschafte hardware, waar zelf extra componenten aan toegevoegd kunnen
>worden zo nodig. De software schrijf je in Processing, met mogelijkheid tot uitbreiding met andere tools en
>omgevingen. Als bewijs van werking, maak je als groep een demonstratievideo, waarin de werking van het
>prototype centraal staat.
>
>Deze opdracht voer je in groepsverband uit.

## Opbouw
De eindopdracht is opgebouwd in Unity. Voor het gebruik van Unity is gekozen omdat hier makkelijk spellen in gebouwd kunnen worden die kunnen worden geëxporteerd naar verschillende besturingssystemen.
Voor het project zal alleen een Android export gemaakt worden. De .apk is te downloaden onder Downloads.
